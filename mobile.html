Getting Started

1

Install the APK

Download the APK above. Go to Settings → Apps → Install unknown apps and allow your browser or file manager. Tap the APK to install.

2

Connect Phantom

Open PEGD Miner and tap Connect Phantom. Phantom will open and ask you to sign a one-time registration message — no transaction fee, no tokens sent.

3

Set threads & start

Pick your CPU thread count (2 is a good start — leaves headroom for normal phone use), then tap START MINING.

4

Receive PEGD

XMR mined by your phone flows to the PEGD treasury via Unmineable. The treasury buys back PEGD and distributes it to miners when earnings reach the payout threshold.

Updating

Check this page for new APK releases. Updates preserve your wallet — just install over the existing app.

Option A — direct install: Download the new APK and tap to install. Android will keep your data.

Option B — via USB (developers): 

adb install -r pegd-miner.apk

The -r flag replaces the existing install without wiping data or disconnecting your wallet.

How It Works

→

xmrig (RandomX)

A native ARM64 binary compiled from source runs as an Android foreground service. It connects to rx.unmineable.com:3333 and mines XMR.

→

Unmineable auto-convert

Unmineable accepts XMR hashrate and pays out in any coin. PEGD uses the SOL treasury address as the payout target with referral code U-FGJ8HA.

→

Phantom identity

Your wallet signs a registration message (no fee) proving ownership. The treasury uses this to attribute mining output to your address for PEGD distribution.
