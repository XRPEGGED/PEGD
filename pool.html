What it mines

  • ⚡ NVIDIA GPU → T-Rex (KawPow and others)
  • ⚡ AMD GPU → lolMiner
  • ⚡ GPU + CPU mine simultaneously
  • 🧠 CPU → XMRig RandomX (runs alongside any GPU)
  • 🪟 Windows 10/11 → automatic Python + desktop shortcut
  • 🔄 Auto-switches GPU coin based on block time
  • 🎯 Auto-tunes intensity for your card
  • 🌙 TOU scheduling (off-peak only)

How $PEGD accumulates

  • 📊 Your hashrate earns share-seconds in the pool
  • 🏦 Mined coins route to Unmineable pool treasury
  • 💸 Unmineable releases SOL to treasury at 0.05 SOL threshold
  • 🔄 Treasury SOL converts to $PEGD via buyback every 3h
  • 👛 $PEGD distributes to miners by share-seconds
  • 🔑 No sign-up — wallet address is your ID
